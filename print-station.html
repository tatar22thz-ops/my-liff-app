ต้องตั้งเครื่องพิมพ์ Zebra ZD888TA เป็นเครื่องพิมพ์เริ่มต้น (Default Printer) ของ Windows ก่อน
แนะนำเปิด Chrome ด้วยแฟล็ก --kiosk-printing เพื่อให้พิมพ์อัตโนมัติโดยไม่ต้องกดยืนยันทีละใบ (คลิกขวาช็อตคัต Chrome → Properties → ต่อท้าย Target ด้วย --kiosk-printing)
โหมดทดสอบ: พิมพ์ทีละ 3 ดวง/แถว (32x25mm ต่อดวง) ตามม้วนตัวอย่างที่มีอยู่ตอนนี้
